# API 명세서: StockMst

## 1\. 개요

`StockMst`는 주식 종목의 현재가와 관련된 상세 데이터를 조회하기 위한 COM 객체입니다.

  - **주요 기능**: 특정 종목 코드에 대한 현재가, 호가, 거래량, 종목 상태 등 다양한 실시간성 데이터를 조회합니다.
  - **관련 CYBOS 화면**: [7021 현재가] 주식 일반
  - **모듈 위치**: `cpdib.dll`

## 2\. 통신 방식

  - **통신 종류**: Request/Reply
  - **연속 데이터 여부**: 연속적이지 않음 (X)

## 3\. Methods

### 3.1. `object.SetInputValue(type, value)`

데이터 요청에 필요한 입력 값을 설정합니다.

| `type` | 입력 데이터 종류 | `value` 데이터 타입 |
| :--- | :--- | :--- |
| **0** | **종목 코드** | `string` |

**예제 코드:**

```python
import win32com.client
inStockMst = win32com.client.Dispatch("dscbo1.StockMst")
inStockMst.SetInputValue(0, "A000660") # SK하이닉스 종목코드 설정
```

### 3.2. `object.Request()` / `object.BlockRequest()`

설정된 종목 코드의 현재가 관련 데이터를 서버에 요청합니다.

  - `object.Request()`: Non-blocking 방식으로 데이터를 요청하며, 결과는 `Received` 이벤트를 통해 수신됩니다.
  - `object.BlockRequest()`: Blocking 방식으로 데이터를 요청하며, 데이터 수신이 완료될 때까지 대기합니다.

**예제 코드:**

```python
# ... SetInputValue 이후 ...
inStockMst.BlockRequest() # 데이터 요청 및 수신 대기
# 데이터 수신 완료 후 GetHeaderValue 호출 가능
```
