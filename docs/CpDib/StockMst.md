Method

object.SetInputValue(type,value)

type에 해당하는 입력 데이터를 value 값으로 지정합니다

type: 입력 데이터 종류

0 - (string) 종목 코드

value: 새로 지정할 값

Python ex)
>>> import win32com.client
>>> inStockMst = win32com.client.Dispatch("dscbo1.StockMst")
>>> inStockMst.SetInputValue(0, "A000660")
